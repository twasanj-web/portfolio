IOS APP · MONEY TRACKING

BUDGIE

A personal finance iOS application designed to help users manage budgets, automate transaction tracking, and understand spending habits through insights and widgets.

App Store GitHub

FEATURE 01

Categorize Your Budget

Users can create spending and fixed categories, allocate budgets and visually track progress for each category.

  • Unlimited categories
  • Budget allocation
  • Progress tracking
  • Category-based spending

FEATURE 02

Automated Payment Tracking

BUDGIE uses Apple Shortcuts integration to help import transactions from SMS messages and organize uncategorized payments.

  • SMS import automation
  • Merchant recognition
  • Custom categorization
  • Payment notifications

FEATURE 03

Performance Insights

Users can view spending performance by day, week or month to better understand their financial habits.

  • Daily insights
  • Weekly reports
  • Monthly analytics
  • Spending charts

FEATURE 04

Widget Experience

BUDGIE provides lock screen widget options that allow users to quickly view their spending information.

  • Lock screen widgets
  • Two widget looks
  • Quick financial overview
  • Personalized experience
